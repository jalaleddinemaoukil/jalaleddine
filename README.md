# Portfolio

Static portfolio site built with Vue 3, Vite, and Vite SSG.

## Requirements

- Node.js 18+
- npm

## Scripts

- `npm install` - install dependencies
- `npm run dev` - start local dev server
- `npm run build` - generate production static build
- `npm run preview` - preview the production build
- `npm run perf:audit` - run Lighthouse CI audit

## Project Structure

- `src/` - app code (pages, components, styles)
- `public/` - static assets
- `api/` - serverless endpoints
- `scripts/` - build and post-processing scripts
- `dist/` - generated production output
